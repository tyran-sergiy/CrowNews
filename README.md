#CrowNews :newspaper: :eyes: 

##### create dynamic news site using:
- HTML, CSS, JavaScript (jQuery), PHP, MySQL
- You can use any existing framework.

-------


##### Site stucture:
- Home page. Contains header, footer, content. In content display three last news
- Realize pagination
- News page. Also contains header, footer. Displays full information about single news
- Search page. Display input to search needed news. Search must be without page refresh.
